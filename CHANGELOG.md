# v0.5 (Current Development)

## Added

- Dedicated SearchView
- Live search
- Search by:
  - Name
  - Address
  - Category
  - Amenities
- Automatic map focus after selecting a search result
- Full-screen search experience

### Improved

- Search UI redesigned
- Frosted glass search field
- Search performance
- Search animations
- Preview Card interaction

### Changed

- Search moved out of MapView into its own screen
- Search now communicates with MapView using callbacks
- Camera animation refined

### Fixed

- Search field interaction bug
- Layering issue preventing keyboard input
- Search presentation bugs
